在线临时聊天室

有时候想跟其他使用者线上文字对话，但又不想把自己的联络资料（例如 Facebook 或即时通讯帐号）泄漏给对方，有些人可能会选择申请一个临时使用的帐号，但这未免也太麻烦！其实只要透过浏览器，即可开启临时、随用即丢的线上聊天室，无须额外下载或安装任何软件。

网站名称：Hack.chat

网站链接：https://hack.chat/

Hack.chat 也将原始码开放，放在 GitHub 上，若你想要自行架设类似服务，可自由使用这套程式码，让每个需要的使用者都能自己架设轻巧且安全的线上聊天室。
使用教学

STEP 1

简单来说，在网址后面加上「?」接上任何一个英文数字组合，就成为一个新的聊天频道，例如：https://hack.chat/?name-of-your-room，把 name-of-your-room 改为任何你想要的名称。当位于同个网址时，才能看到其他人的对话内容，此外，Hack.chat 没有管理机制，也就是它无法对其他人进行禁言、踢除等管理模式。

STEP 2

开启任一聊天频道后，会显示要求使用者输入暱称（Nickname）的提示讯息。

另外，Hack.chat 对于中文字可正常显示，当有人加入或离开对话时，聊天室会以绿色的文字来进行提示，后加入的使用者无法读取之前的对话内容，因为 Hack.chat 不会保留任何的对话记录。

值得一试的三个理由：

1.开放原始码，可自由下载、架设聊天室伺服器

2.提供高安全性的线上对话环境，无须预先下载任何软件

3.没有使用限制，对于中文支援良好
